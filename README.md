# 晚安Android

Wanandroid Flutter App实现版本

## 项目说明

~~~~
├── README.md           # 项目介绍
├── android             # Android
├── assets
│   ├── fonts           # 字体文件
│   ├── images          # 图片
│   └── jsons           # json文件
├── ios                 # iOS
├── lib
│   ├── api             # 接口管理
│   ├── common          # 一些工具类，如通用方法类、网络接口类、保存全局变量的静态类等
│   ├── i18n            # 本地化文件
│   ├── main.dart       # 入口文件
│   ├── models          # 接口对应的Dart Model类会在此目录下
│   ├── routes          # 存放所有路由页面类
│   ├── states          # 保存APP中需要跨组件共享的状态类
│   ├── utils           # APP内工具类
│   └── widgets         # APP内封装的一些Widget组件都在该目录下
├── pubspec.yaml        # Flutter依赖管理
├── test                # 测试文件      
~~~~
